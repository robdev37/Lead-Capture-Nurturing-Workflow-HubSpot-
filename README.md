# 🚀 Lead Capture & Nurturing Workflow (HubSpot)

## 📌 Overview
This project demonstrates an end-to-end **lead capture and email automation system** built using HubSpot.

The workflow automatically captures leads from a form, assigns them to an owner, sends follow-up emails, and creates tasks for sales — ensuring no lead is missed.

---

## ⚙️ Workflow Summary
The automation follows a structured lead nurturing sequence:

1. Lead submits form (Trigger enrollment)  
2. Contact is assigned to a sales owner (round-robin)  
3. Contact record is created/updated in CRM  
4. Immediate confirmation email is sent  
5. Delay (time for engagement)  
6. Follow-up email #1 is sent  
7. Delay  
8. Final follow-up email is sent  
9. Task is created for sales team follow-up  
10. Contact record is updated  

---

## 🎯 Key Features
- Automated lead capture from forms  
- Email nurturing sequence (3-step follow-up)  
- Task creation for manual follow-up  
- CRM record updates for tracking  
- Fully automated workflow (no manual intervention needed)
